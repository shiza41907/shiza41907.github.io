Describe the Internet Protocol Suite (TCP/IP) model of networking, in general terms, outlining how this protocol provides the data transfer mechanism required to establish client/server relationships.

When computers connect and transmit data between each other, they have to follow a set of rules to do so. These rules are made unviersally so that all computers can use them. These rules are called protocols. There are many different protocols made for different purposes. 'Internet Protocol Suite' is the term we use to refer to the protocols all together. The two most important protocols are Transmission Control Protocol (TCP) and the Internet Protocol (IP), which helps in making sure that data is delivered at the correct place without any errors.

How is the Hypertext Transfer Protocol (HTTP) used to facilitate client/server interaction?

The HTTP client (a browser is an example) intiates a HTTP request towards the server. Once the request is made,the client waits for the response. The server then processes the request by displaying the requested objects. This means that if the client makes a request for HTML pages towards the server, then the server will request the process by displaying the requested HTML pages.
